# Archive Folder

This folder contains archived files that are not actively used by the application but are kept for reference.

## Folder Structure

### `/old-docs`
- Historical documentation
- Backup files
- Old planning documents
- PDF exports

### `/console-logs`
- Browser console logs from debugging sessions
- Chrome logs
- Development console outputs
- These can be safely deleted periodically

### `/temp-files`
- Temporary scripts
- Debug utilities
- One-off test files
- Diagnostic scripts

## Note
These files are not required for the application to run and are kept for historical reference only. They can be safely deleted if space is needed.
